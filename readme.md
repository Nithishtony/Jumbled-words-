# 🔤 Jumbled Words

A simple and fun Python console game where players guess the correct
word from a shuffled version. This project is designed for beginners to
practice core Python concepts like strings, loops, conditionals, and
randomness.

------------------------------------------------------------------------

## 📖 Overview

**Jumbled Words** is a beginner-friendly Python project that:

-   Selects a random word
-   Shuffles its letters
-   Prompts the user to guess the original word
-   Validates the answer and displays feedback

It's a great starter project for improving logical thinking and Python
basics.

------------------------------------------------------------------------

-   🔀 Random word jumbling

## 🚀 Installation

### 1️⃣ Prerequisites

Make sure Python is installed:

``` bash
python --version
```

If not installed, download from: https://www.python.org/downloads/

------------------------------------------------------------------------

### 2️⃣ Clone the Repository

``` bash
git clone https://github.com/Nithishtony/Jumbled-words-.git
cd Jumbled-words-
```

------------------------------------------------------------------------

## ▶️ Usage

Run the program using:

``` bash
python main.py
```

Follow the on-screen instructions to play the game.
