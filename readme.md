# 🔤 Jumbled Words

A simple and fun Python project that generates or solves jumbled words.  
This project demonstrates basic string manipulation, logic building, and problem-solving using Python.

---

## 📌 Features

- 🔀 Jumbles words randomly
- 🧠 Allows users to guess the correct word
- ✅ Checks user input and gives feedback
- 🎯 Beginner-friendly Python project
- ⚡ Lightweight and easy to run

---

## 🛠️ Tech Stack

- **Language:** Python
- **Concepts Used:**  
  - Strings  
  - Random module  
  - Loops and conditionals  
  - User input handling
